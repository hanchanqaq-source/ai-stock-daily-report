# Web-P22 持仓预览 mock-only

## 1. 目标

Web-P22 在 `mock-only-preview` 独立入口中新增“持仓预览”静态区域，将模块范围列表里的“持仓预览”从“后续建设”升级为“可预览”，并提供同页锚点入口 `#mock-portfolio-preview`。

本功能服务于“股票基金质量分析系统”的本地页面渲染检查，页面显示名称保持为“AI股票基金每日信息报告”。

## 2. 页面结构

持仓预览位于现有 mock-only 安全预览页面内，不引入 React Router，不引入新依赖，也不接入正式 Web App 路由。

页面结构包括：

1. 模块范围区域中的“持仓预览”卡片：
   - 状态显示为“可预览”。
   - 提供“进入预览”链接。
   - 链接指向当前页面锚点 `#mock-portfolio-preview`。
2. 持仓预览内容区域：
   - 标题：`持仓预览`。
   - 副标题说明只展示静态脱敏 fixture。
   - 显示安全标签：`模拟数据`、`REDACTED FIXTURE DATA`、`非真实账户`、`非投资建议`、`不会发送通知`、`不会交易`。
   - 显示账户总览、持仓列表、风险提示和今日观察备注。

## 3. fixture 数据说明

持仓预览使用 TypeScript 内置静态脱敏 fixture，不读取真实账户、真实基金、真实行情或真实交易记录。

账户总览 fixture：

- 模拟账户：本地预览组合
- 模拟持仓总额：¥59,167.78
- 模拟目标仓位：¥180,000.00
- 模拟仓位比例：32.9%

持仓列表 fixture：

- 硬科技观察仓：¥14,879.70，占比 25.1%，模拟浮动 +18.33%，风险 中高
- AI 硬件观察仓：¥6,877.54，占比 11.6%，模拟浮动 -0.46%，风险 中
- 半导体观察仓：¥2,115.71，占比 3.6%，模拟浮动 -4.01%，风险 中高
- 海外科技观察仓：¥2,292.78，占比 3.9%，模拟浮动 -1.61%，风险 中

这些名称、金额、比例、盈亏和风险等级只用于页面渲染测试，不对应真实基金、真实账户或正式投资判断。

## 4. 安全边界

Web-P22 必须保持以下边界：

- mock-only。
- 仅用于本地安全预览。
- 不启动后端。
- 不自动打开浏览器。
- 不接真实 API。
- 不接 provider。
- 不接通知。
- 不读取环境配置文件。
- 不读取密钥类配置。
- 不发送日报。
- 不进行交易。
- 不读取真实账户。
- 不读取真实基金平台。

本功能不是正式日报，不是投资建议，不会发送通知，也不会交易。

## 5. 本地验证方法

在 `apps/dsa-web` 下运行 mock-only 相关轻量测试：

```bash
npm run test -- tests/mocks/preview/mockOnlyPreview.test.ts
npm run test -- tests/mocks/preview-entry/mockOnlyPreviewEntry.test.ts
npm run test -- tests/mocks/preview/mockOnlyPreviewNetworkBoundary.test.ts
npm run build
```

如需验证 Windows safe preview 脚本静态边界，可继续运行仓库中对应 Python 静态测试。

## 6. 回滚方式

如需回滚 Web-P22，可移除持仓预览 fixture、页面渲染区、相关测试断言和本说明文档，并将模块范围中的“持仓预览”状态恢复为“后续建设”。
