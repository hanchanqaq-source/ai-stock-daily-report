# 股票基金双中心 Build E7：当前用户本机持久化

当前用户选择是股票中心、基金中心共用的低敏本机偏好。Build E7 将该选择写入 SQLite 的独立偏好表，重启便携版后恢复到同一位用户；股票和基金持仓仍继续按该用户、按领域隔离。

## 安全与兼容

- 新表为加法式变更，既有用户、股票持仓、基金持仓、历史和恢复点不改写、不清空。
- 新增 migration 记录；旧数据库首次启动会创建偏好表后写入记录。
- 仅允许 localhost API 设置当前用户；不存在的用户被拒绝。
- 删除当前用户时仅把当前选择回退到“本人”，不影响其他用户或系统设置。
- 不保存密钥、账户、交易、截图、日志或 Provider 数据。

## 验证

- 切换到非默认用户后重启后端，当前用户仍为该用户。
- 删除该用户后，当前用户安全回退为“本人”。
- 既有股票和基金持仓的按用户、按领域隔离测试继续通过。
