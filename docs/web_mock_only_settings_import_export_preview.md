# Web-P20 mock-only 设置与导入导出预览

## 目标

Web-P20 在 Windows localhost-only safe preview 中增加一个只读状态面板，帮助用户理解后续设置、导入和导出的安全边界。

## 当前范围

- 只在 `mock-only-preview` 独立入口展示。
- 固定为 mock-only 本地预览。
- 只展示设置锁定、人工复核和不生成备份三种状态。
- 不接入真实 `SettingsPage`、真实 API、路由或后端。

## 明确不做

- 不读取 `.env`、本地文件或剪贴板。
- 不导入或导出配置。
- 不生成或下载备份。
- 不保存 Token、API Key、Webhook、真实账户或个人金额。
- 不启动后端、不发送通知、不请求真实网络。

## 后续真实功能前提

真实设置或导入导出能力必须在 private 配置隔离、明确预览清单、用户确认、回滚方案和独立安全复核完成后另行设计；不得复用本 mock-only 面板直接执行操作。
