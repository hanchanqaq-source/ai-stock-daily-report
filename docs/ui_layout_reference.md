# Web 页面布局参考模型说明

## 1. 参考 UI 的优点

参考 UI 的主要优点是左侧导航清晰、顶部搜索明显、点位卡片直观、资讯区域顺手。后续页面可以吸收这些信息组织方式，但不照抄具体样式或交互。

## 2. 本项目改造方向

本项目的页面参考模型会结合账户、基金、股票、市场指数、行情 / 净值摘要、风险雷达、个人记录入口等能力，形成适合个人市场工作台的结构。重点增强方向包括账户切换、基金 / 股票分流、市场指数矩阵、持有 vs 收藏、数据来源状态和个人记录入口。

## 3. 页面结构

页面参考模型包括：

- 左侧导航：承载首页、账户、自选 / 观察、持有、基金、股票、市场指数、点位卡片、资讯、设置等入口。
- 顶部区域：保留搜索、账户切换、市场切换和数据来源状态。
- 主区域卡片：展示账户总览、行情 / 净值摘要、关联板块、个人观察点位卡片、风险雷达、相关新闻 / 公告和个人记录入口。

## 4. 点位卡片区域

点位卡片区域后续用于渲染观察买入区、二次关注区、风险控制位、目标观察区。该区域只表达个人观察和记录，不代表系统自动执行任何操作，也不代表真实前端已经实现。

## 5. 后续 Web 路线

- Web-P15：实时行情与基金净值看板。
- Web-P16：个人观察点位卡片页面。
- Web-P17：账户首页综合看板。
