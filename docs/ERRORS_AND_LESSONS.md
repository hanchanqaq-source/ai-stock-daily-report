# Errors and Lessons

- PR95-FIX-01：安全词上下文误伤。页面层、demo payload 层、文档层不能使用同一套粗暴关键词规则；个人工作台允许资金看板、买入观察、实时涨跌等直观词语；真正禁止的是强制交易、自动执行、收益承诺和真实敏感值泄露。
- PR95-FIX-02：页面展示层不要使用容易被误判为系统代办的状态词，统一改为“记录状态 / 观察记录”。个人工作台可以使用直接易懂的观察词；测试应区分内部字段名和页面展示文案，不应使用粗暴关键词断言。
- PR95-FIX-03：记录状态测试契约同步。`auto_execute` 可作为内部兼容字段存在，但页面展示应使用“记录状态 / 观察记录”；测试只检查页面展示词，不把内部字段名等同于页面承诺。
