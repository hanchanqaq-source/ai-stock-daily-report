# Windows Portable-M2 更新规划

## 当前阶段：M2.1（只校验）

Portable-M2.1 仅允许用户手动选择 Portable ZIP 与对应的 `.sha256` 文件，应用只读计算 ZIP 的 SHA-256 并显示“通过”或“不通过”。

- 不下载文件。
- 不解压、替换、覆盖、删除或重启程序。
- 不修改 `data/`、`config/`、`logs/`、`plugins/` 或任何持仓数据。
- 仅便携版在设置页显示“校验便携更新包”；安装版的检查/安装更新按钮不调用此能力。

M2.1 暂不检查 ZIP 内的中文 EXE、打包后的 backend EXE 或目录结构；这些属于下一步只读结构检查，不构成更新安装。

## 后续阶段：M2.2（尚未实现）

真正替换更新必须在用户关键授权后执行，并依次完成：更新前恢复点、临时解压验证、仅替换程序文件、保留 `data/config/logs/plugins`、启动验证与失败回退。M2.2 前不得复用 M2.1 的校验入口执行任何写操作。
